## 💬 Description 💬

You will be given a lower case string and your task will be to remove k characters from that string.

## 🥚 Input 🥚

```
"abracadabra" (String)
```

## 🐣 Output 🐣

```
"bracadabra" (String)
```

## 👓 Explanation 👓

```

 2# "_bracadabra"
 3# "bracadabra" 🎉
```

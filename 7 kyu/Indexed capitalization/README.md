# Indexed capitalization - 7kyu

The solution to this problem is written in JavaScript the detail about the problem is below and you can check the solution above.

## 🥚 Input 🥚

```
('abcdef', [1, 2, 5]) (String, Array of Numbers)
```

## 🐣 Output 🐣

```
"aBCdeF" (String)
```

## 👓 Explanation 👓

```
1# ('abcdef', [1, 2, 5])
2# (['a', 'b', 'c', 'd', 'e', 'f'])
3# (['a', 'B', 'C', 'd', 'e', 'F'])
4# "aBCdeF"  🎉
```

## 💬 Description 💬

Create a function that returns the sum of the two lowest positive numbers given an array of minimum 4 positive integers. No floats or non-positive integers will be passed.

## 🥚 Input 🥚

```
[19, 5, 42, 2, 77] (Array of Numbers)
```

## 🐣 Output 🐣

```
7 (Number)
```

## 👓 Explanation 👓

```
 1# [19, 5, 42, 2, 77]
 2# [2, 5, 19, 42, 77]
 3# [2, 5]
 4# 2 + 5
 5# 7 🎉
```

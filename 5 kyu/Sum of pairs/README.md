# Sum of pairs - 5kyu

The solution to this problem is written in JavaScript the detail about the problem is below and you can check the solution above.

## 🥚 Input 🥚

```
([10, 5, 2, 3, 7, 5], 10) (Array of Numbers, Number)
```

## 🐣 Output 🐣

```
[3, 7] (Array of Numbers)
```

## 👓 Explanation 👓

```
 1# [10, 5, 2, 3, 7, 5]
 2# [10, 5, 2, "3", "7", 5]
 3# 3 + 7 = 10
 4# [3, 7] 🎉
```

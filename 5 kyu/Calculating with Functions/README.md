# Calculating with Functions - 5kyu

The solution to this problem is written in JavaScript the detail about the problem is below and you can check the solution above.

## 🥚 Input 🥚

```
seven(times(five())) (Expression)
```

## 🐣 Output 🐣

```
35 (Number)
```

## 👓 Explanation 👓

```
 1# seven(times(five()))
 2# seven(times(5))
 3# seven(*(5))
 4# 7(*(5))
 5# 7*5
 6# 35 🎉
```

## 💬 Description 💬

The function should take three arguments - operation(string/char), value1(number), value2(number).
The function should return result of numbers after applying the chosen operation.

## 🥚 Input 🥚

```
('+', 4, 7) (String, Number, Number)
```

## 🐣 Output 🐣

```
11 (Number)
```

## 👓 Explanation 👓

```
 1# ('+', 4, 7)
 2# (4 + 7)
 3# 11 🎉
```

# FILLING AN ARRAY (PART I) - 8kyu

👨🏻‍💻 Solution using Javascript 👨🏻‍💻

## Description

Write a function that produces an array with the numbers 0 to N-1 in it.

## Input

```
🥚 5 🥚 (Integer)
```

## Output

```
🐣 [0, 1, 2, 3, 4] 🐣 (Array - Integers)
```

## Explanation

```
 1# 5
 2# [1,2,3,4,5]
 3# [0,1,2,3,4]🎉
```

## Note

⚠ If there is no parameters create an empty array ⚠

## 💬 Description 💬

If there is enough space, return 0, and if there isn't, return the number of passengers he can't take.

## 🥚 Input 🥚

```
(10, 5, 5) (Number, Number, Number)
```

## 🐣 Output 🐣

```
0 (Number)
```

## 👓 Explanation 👓

```
1# (10, 5, 5)
2# (10, (5 + 5))
3# ((5 + 5) <= 10)
4# 0 🎉
```

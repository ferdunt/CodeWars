## 💬 Description 💬

Your task is to write a function that takes a String and returns an Array/list with the length of each word added to each element .

## 🥚 Input 🥚

```
"apple ban" (String)
```

## 🐣 Output 🐣

```
["apple 5", "ban 3"] (Array of Strings)
```

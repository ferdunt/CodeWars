# Irreducible Sum of Rationals - 6kyu

The solution to this problem is written in JavaScript the detail about the problem is below and you can check the solution above.

## 🥚 Input 🥚

```
[[1, 2],[1, 3],[1, 4]] (Array of Arrays)
```

## 🐣 Output 🐣

```
[13, 12] (Array)
```

## 👓 Explanation 👓

```
 1# [[1, 2],[1, 3],[1, 4]]
 2# [(1/2), (1/3), (1/4)]
 3# [(1/2) + (1/3) + (1/4)]
 4# [(5/6) + (1/4)]
 5# [(26/24)]
 6# [(13/12)]
 7# [13, 12] 🎉
```

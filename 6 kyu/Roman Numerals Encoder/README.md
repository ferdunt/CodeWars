# Roman Numerals Encoder - 6kyu

The solution to this problem is written in JavaScript the detail about the problem is below and you can check the solution above.

## 🥚 Input 🥚

```
1990 (Number)
```

## 🐣 Output 🐣

```
"MCMXC" (String)
```

## 👓 Explanation 👓

```
 1# 1990
 2# 1000 = M, 900 = CM, 90 = XC
 3# "MCMXC" 🎉
```

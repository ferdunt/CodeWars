# Equal Sides Of An Array - 6kyu

The solution to this problem is written in JavaScript the detail about the problem is below and you can check the solution above.

## 🥚 Input 🥚

```
[1, 2, 3, 4, 3, 2, 1] (Array of Numbers)
```

## 🐣 Output 🐣

```
3 (Number)
```

## 👓 Explanation 👓

```
 1# [1, 2, 3, 4, 3, 2, 1]
 2# [[1, 2, 3], 4, [3, 2, 1]]
 3# [[0, 1, 2], 3, [4, 5, 6]]
 4# 3 🎉
```
